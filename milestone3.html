⚡ IO Rodeostat

Implemented by Kayla. We are using the IO Rodeo’s Rodeostat as our potentiostat circuit base.

🔗 Rodeostat Product Page

How to set up IO Rodeostat:
  • Download the potentiostat.py library from the IO Rodeo GitHub page (version 0.0.9).
  • Follow the setup instructions: Installation Guide
  • Customize your code and start using your Rodeostat!
Methodology:
  • 8 second quiet time for glucose reaction on test strip
  • Apply 200 mV constant potential step for 2 seconds
  • Run linear regression of i vs 1/√t and convert slope to glucose concentration
UI Integration:
  • Test strip detection (operational but not currently implemented)
  • Analyte detection (implemented)

The Rodeostat is an open source potentiostat designed to work with the Adafruit ItsyBitsy M4 Express. It supports multiple output voltage and current ranges.
