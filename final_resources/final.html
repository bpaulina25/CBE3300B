Throughout the project, the Arduino GIGA R1 WiFi board and its companion display have served as our main user interface. Our work focused on enabling the display of real-time glucose concentration values received via serial communication with a Python script. Key features of the final GIGA display implementation include:

  • Live concentration display: Values from CSV files are sent from Python to Arduino and rendered as on-screen readings.
  • Touchscreen interactivity: The UI includes user selection via checkboxes and the ability to add new users with an on-screen keyboard.
  • Concentration logging: Readings are tagged with user names and transmitted back to Python for structured storage in CSV format.
  • Improved font scaling and UI styling using LVGL for clarity and readability on the 800x480 display.
  • Integration with potentiostat functionality in Milestone 4 to support glucose readings derived from chronoamperometry.

This interface was iteratively developed through multiple milestones, progressively incorporating user input handling, automated display refreshes, and modular file organization across the Arduino and Python environments. It is now a fully functional frontend for our glucometer system.
