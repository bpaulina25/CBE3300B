🖥️ Arduino GIGA Display Overview

Throughout the project, the Arduino GIGA R1 WiFi board and its companion display have served as our main user interface. Our work focused on enabling the display of real-time glucose concentration values received via serial communication with a Python script. Key features of the final GIGA display implementation include:

  • Live concentration display: Values from CSV files are sent from Python to Arduino and rendered as on-screen readings.
  • Touchscreen interactivity: The UI includes user selection via checkboxes and the ability to add new users with an on-screen keyboard.
  • User system: A dynamic list of users can be selected before each reading. Selecting "New User" prompts an on-screen keyboard to register a new entry. User-specific data is then saved alongside the readings.
  • Reading system: The system remains in a waiting state until it receives a valid signal from Python, indicating that a glucose reading is available. Once the signal is received, the value is parsed, formatted, and displayed instantly.
  • Display refresh and double-tap navigation: A built-in refresh logic avoids flickering and ensures data integrity. Users can double-tap to return to the home screen or to confirm user input transitions.
  • Concentration logging: Each reading is tagged with the selected user and transmitted back to the Python environment for structured storage in a CSV file.
  • Improved font scaling and UI styling: LVGL is used to style the display with larger fonts, contrasting labels, and appropriate spacing for enhanced readability on the 800x480 display.
  • Integration with potentiostat functionality: The Arduino interface also works with data collected from the Rodeostat-based chronoamperometry setup developed in Milestone 4.

This interface was iteratively developed through multiple milestones, progressively incorporating user input handling, automated display refreshes, and modular file organization across the Arduino and Python environments. It is now a fully functional frontend for our glucometer system.
