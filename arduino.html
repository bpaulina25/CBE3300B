ðŸ“Œ Setup Instructions

  • Download Arduino Mbed OS Edge Boards: Go to boards, search for it, and install.
  • Edit the Library: Search for **Arduino Giga Display**, click install, and select **Install All**.
  • Change the board to: Arduino Giga R1.
  • Test Board Connection: Upload the **RGB LED Test** example.
  • Test the Touch Screen:
    • Use Touch_IRQ (triggers an action when touched).
    • We are NOT using: Touch_polling (which constantly asks for a touch event).
  • Test LCD Screen:
    • Using **Arduino Logo Example**, displays an image on the screen.
    • Using **Arduino Logo Drawing**, recreates the image using primitives (circles & rectangles).
    • Using **LVG Demo**, shows an interactive UI (buttons, clickers, drag bars, loading bar).
  • Convert an Image to C Array: Use **BinaryRGB565** format and adjust the size before converting.
